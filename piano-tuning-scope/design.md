# Piano Tuning Scope - 모바일 앱 인터페이스 디자인

## 설계 철학

PT-100 조율시험기의 실제 출력물과 동일한 레이아웃을 디지털로 재현하여, 조율사에게 친숙하고 즉시 사용 가능한 전문 도구 느낌을 제공한다. 모바일 세로 모드(9:16)에서 한 손으로 조작 가능하도록 최적화한다.

---

## Screen List

### 1. Home (튜너 메인 화면)
- 앱의 핵심 화면. 마이크 감지, 스트로브 표시, 현재 음 정보를 한 눈에 보여줌
- 상단: 현재 감지 음 + 센트 오차 표시 (대형 숫자)
- 중단: 스트로브 튜너 (PT-100 스타일 빨간 바 애니메이션)
- 하단: 마이크 시작/정지 버튼 + 자동저장 토글

### 2. Chart (조율 커브 화면)
- PT-100 출력물과 동일한 조율 커브 그래프
- X축: 88건반 (피아노 건반 그림 포함)
- Y축: -40 ~ +40 센트
- 상/하한 허용 커브 (계단식 S자)
- 측정된 점 표시 (범위 내: 네이비, 범위 외: 빨강)
- 핀치 줌/패닝 지원

### 3. Sessions (세션 관리 화면)
- 저장된 세션 목록 (최대 10개)
- 세션 생성/삭제/이름 변경
- 각 세션의 측정 진행률 표시
- 세션 선택 시 해당 데이터로 전환

### 4. Settings (설정 화면)
- 안정 대기 시간 조절 (0.5~3.0초)
- 기준음 재생 (A4 = 440Hz)
- 맥놀이(Beat) 재생 (1~10Hz)
- 내보내기 (PDF/이미지)
- 성명 입력 (내보내기용)

## Primary Content and Functionality

### Home 화면
| 영역 | 콘텐츠 | 기능 |
|------|---------|------|
| 상단 바 | 세션명, 진행률 | 세션 전환 |
| 음 표시 | 음이름(A4), 건반번호(49), 주파수(440.0Hz) | 실시간 업데이트 |
| 센트 미터 | 큰 숫자 (+2.3¢), 수평 바 | 색상으로 정확도 표시 |
| 스트로브 | PT-100 스타일 빨간 바 애니메이션 | 멈추면 정확, 흐르면 오차 |
| 컨트롤 | 마이크 버튼, 자동저장 토글, 되돌리기 | 터치 조작 |

### Chart 화면
| 영역 | 콘텐츠 | 기능 |
|------|---------|------|
| 그래프 | SVG 조율 커브 | 핀치 줌, 패닝 |
| 건반 | 하단 피아노 건반 시각화 | 측정된 건반 하이라이트 |
| 통계 | 측정 수, 평균 오차, 최대/최소 | 실시간 업데이트 |
| 내보내기 | PDF/이미지 버튼 | 공유 시트 |

### Sessions 화면
| 영역 | 콘텐츠 | 기능 |
|------|---------|------|
| 리스트 | 세션 카드 (이름, 날짜, 진행률) | 탭하여 선택 |
| 액션 | 새 세션 버튼 | 생성 |
| 스와이프 | 삭제/이름변경 | 스와이프 액션 |

## Key User Flows

### 조율 측정 플로우
1. 앱 실행 → Home 화면
2. 세션 선택 또는 새 세션 생성
3. 마이크 시작 버튼 탭
4. 피아노 건반 누름 → 음 감지 → 센트 표시
5. 자동저장 ON: 0.8초 안정 후 자동 저장
6. 자동저장 OFF: 수동 저장 버튼 탭
7. Chart 탭으로 이동하여 커브 확인

### 내보내기 플로우
1. Chart 화면에서 내보내기 버튼 탭
2. 성명 입력 (선택)
3. PDF 또는 이미지 선택
4. 시스템 공유 시트로 저장/공유

### 스트로브 정밀 측정 플로우
1. 마이크 활성화 상태에서 건반 누름
2. 스트로브 바 움직임 관찰
3. 기준값 ±1/±10 조정하여 바 멈춤
4. "안정값으로 저장" 버튼 탭

## 모바일 최적화 원칙

1. 한 손 조작: 주요 버튼은 하단 thumb zone에 배치
2. 큰 터치 타겟: 최소 44x44pt
3. 실시간 피드백: 햅틱 + 시각적 피드백
4. 화면 꺼짐 방지: 마이크 활성 시 자동
5. 세로 모드 전용: 조율 작업 시 안정적
